To add a link to your Pinterest profile, open **Settings → Edit profile** and paste your URL into the **Website** field. For the best reach, also **claim your website** in settings so your profile shows a verified link and you get analytics on Pins that drive to it.

Pinterest only shows one profile link, so if you want to send people to more than one destination - a shop, a blog, a portfolio, a newsletter - link to a link-in-bio page. That single URL becomes a hub you can update whenever you publish something new.

## Step by step

1. Build a free link-in-bio page with a block for each link.
2. Copy your URL, e.g. `lin.ky/yourname`.
3. In Pinterest, paste it into **Edit profile → Website**.

Because the same link works on every platform, you can use it on Pinterest, Instagram, and TikTok at once. Start [free](/i/pricing), and add live blocks like your [latest posts](/i/integrations/instagram) to keep the page fresh.
